# 🧮 Modern Calculator

A modern, responsive, and accessible calculator built using **HTML**, **CSS**, and **JavaScript**.

## ✨ Features

- ➕ Basic arithmetic operations
  - Addition
  - Subtraction
  - Multiplication
  - Division
  - Percentage

- 🔢 Decimal calculations

- 🔄 Consecutive calculations

- 📜 Calculation history

- 🌙 Dark / Light mode

- ⌨️ Keyboard shortcuts

- 📋 Copy result to clipboard

- 📱 Fully responsive design

- ⚡ Smooth animations

- 🛡️ Error handling

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)

## 🚀 How to Run

### Option 1

Download or clone the repository and open `index.html` in your browser.

### Option 2

Using Python:

```bash
python -m http.server 8000
```

Then open:

```
http://localhost:8000
```
